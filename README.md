# Numberwang.js

That's NumberWang!

# Example
```javascript
var Numberwang = require('numberwang-js');
Numberwang(3); // false
Numberwang(9); // false
Numberwang(16); // false
Numberwang(12); // false
Numberwang(8.2); // false
Numberwang(4); // true - That's NumberWang!
```

# Install

`npm install numberwang-js`
